# fungal-priming-protocol
> Open-Source Multi-Factor Protocol for Accelerating Fungal Biodegradation of Synthetic Polymers.

[![License: CC BY 4.0](https://img.shields.io/badge/License-CC_BY_4.0-lightgrey.svg)](https://creativecommons.org/licenses/by/4.0/)

## Overview
This repository hosts the official standalone open-source scientific framework dedicated to compressing the timescales of synthetic polymer bioremediation (PE, PS, PP, PET). The protocol synchronizes abiotic pretreatments, localized microcurrent stimulation, bacterial-fungal consortia, and sub-toxic monomer enzyme priming into a unified, testable laboratory workflow.

Additionally, this project advocates for the **"Mycelium Compatibility Test"** — a conceptual standard to validate the biocompatibility of novel synthetic materials prior to industrial mass production.

## Repository Architecture
